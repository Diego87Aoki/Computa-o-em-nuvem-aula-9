# ☁️ Computação em Nuvem – Aula 9 🚀

Bem-vindo(a) ao repositório da **Aula 9 de Computação em Nuvem**!  
Aqui você vai encontrar os materiais, exercícios e exemplos práticos que exploram conceitos modernos da nuvem de forma leve e direta 😎✨

---

## 🧭 **Sobre o Projeto**

Este repositório foi criado como parte da disciplina **Computação em Nuvem**, ministrada pela **Professora Simone Tatiane do Canto**, na  
📍 **Faculdade Anhanguera – Unidade 4 Ouro Verde – Campinas/SP**  
💻 **Curso:** Análise e Desenvolvimento de Sistemas  
📚 **Semestre:** 5º  

O objetivo é organizar e compartilhar os conteúdos desenvolvidos em aula, servindo como apoio para estudos, revisões e prática de conceitos importantes de computação em nuvem 🌐⚡

---

## 🧰 **Tecnologias e Ferramentas**

- ☁️ **Serviços em Nuvem** (ex: AWS, Azure, GCP)  
- 🐳 **Containers** e orquestração básica  
- ⚡ **Serverless / FaaS** (Função como Serviço)  
- 🕸️ **Serviços de Rede** e instâncias virtuais

---

## 🚀 **Como Usar**

Siga os passos abaixo para explorar os conteúdos 👇

1. **Clone este repositório**  
   ```bash
   git clone https://github.com/Diego87Aoki/computacao-em-nuvem-aula-9.git
````

2. **Acesse a pasta do projeto**

   ```bash
   cd computacao-em-nuvem-aula-9
   ```

3. **Siga as instruções em cada pasta**
   Cada exercício possui um README ou arquivos com explicações passo a passo 📝

4. **Execute os exemplos**
   Você poderá:

   * Rodar simulações locais 🧪
   * Subir funções serverless 🛰️
   * Testar containers 🐳
   * Explorar conceitos de rede em nuvem 🌐
